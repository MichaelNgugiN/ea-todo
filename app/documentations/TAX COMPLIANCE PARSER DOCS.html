INTRODUCTION
This documentation will describe how to set-up and run the tax complieance parser. There are the steps taken to run the application after unzipping to a folder in your computer. 

The document will include two parts;

How to run the application and parse the tax compliance pdf file to csv.

A source code documentation on the algorithm used.


Assumptions

No restrictions on portable software on the machine. The application is a portable software and can run on any USB, Network or Hardrive without prior installation. 

The pdf totals need to be completely filled with trailing two daecimal places. (IMPORTANT). Otherwise the software will not parse correctly. Comma separated total figures can be parsed.

The Program is purely writen in Javascript (NodeJS) and Packaged with electron. The parser used is Pdfreader, From Link to website.

Introduction
This documentation will describe how to set-up and run the tax complieance parser. There are the steps taken to run the application after unzipping to a folder in your computer. 

The document will include two parts;

How to run the application and parse the tax compliance pdf file to csv.

A source code documentation on the algorithm used.


Assumptions

No restrictions on portable software on the machine. The application is a portable software and can run on any USB, Network or Hardrive without prior installation. 

The pdf totals need to be completely filled with trailing two daecimal places. (IMPORTANT). Otherwise the software will not parse correctly. Comma separated total figures can be parsed.

The Program is purely writen in Javascript (NodeJS) and Packaged with electron. The parser used is Pdfreader, From Link to website.

Introduction

What is Vue.js?

Vue (pronounced /vjuː/, like view) is a progressive framework for building user interfaces. Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable. The core library is focused on the view layer only, and is very easy to pick up and integrate with other libraries or existing projects. On the other hand, Vue is also perfectly capable of powering sophisticated Single-Page Applications when used in combination with modern tooling and supporting libraries.

If you are an experienced frontend developer and want to know how Vue compares to other libraries/frameworks, check out the Comparison with Other Frameworks.

Getting Started

The official guide assumes intermediate level knowledge of HTML, CSS, and JavaScript. If you are totally new to frontend development, it might not be the best idea to jump right into a framework as your first step - grasp the basics then come back! Prior experience with other frameworks helps, but is not required.
